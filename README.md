# Vision Trace AI - Lost Object Finder

## Overview

Vision Trace AI is an AI-powered Lost Object Finder that helps users locate misplaced objects using computer vision and deep learning techniques. The system uses YOLOv8 for object detection and object matching algorithms to identify and locate objects from uploaded images or camera input.

## Features

* Real-time object detection using YOLOv8
* Object matching and identification
* Camera-based object search
* Image upload support
* User-friendly web interface
* Fast and accurate object localization

## Technologies Used

* Python
* Django
* OpenCV
* YOLOv8 (Ultralytics)
* PyTorch
* HTML
* CSS

## Project Structure

* config/ - Django project configuration
* detector/ - Core object detection and matching modules
* templates/ - Frontend HTML templates
* media/ - Sample images and outputs

## Installation

1. Clone the repository
2. Install dependencies:

pip install -r requirements.txt

3. Run the server:

python manage.py runserver

4. Open the browser and access the application.
